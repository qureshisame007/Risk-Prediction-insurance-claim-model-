# 🛡️ Insurance Claim Risk Prediction using Neural Network (PyTorch)

## 📌 Overview

This project predicts whether a customer is likely to file an insurance claim using a Feedforward Neural Network built with PyTorch.

## 🚀 Features

- Binary Classification
- Neural Network using PyTorch
- Data Preprocessing
- StandardScaler
- TensorBoard Logging
- Model Saving

## 🛠 Technologies

- Python
- PyTorch
- Pandas
- NumPy
- Scikit-learn
- TensorBoard
- Matplotlib

## 📊 Dataset Features

- Age
- BMI
- Annual Income
- Deductible
- Coverage Limit
- Blood Pressure
- Cholesterol
- Stress Score
- Premium Paid
- Vehicle Age

Target:

claim_flag

0 = No Claim

1 = Claim
